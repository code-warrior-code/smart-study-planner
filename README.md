# Smart Study Planner

An AI-assisted study planner that helps students build and adjust exam prep schedules in real time.

## What it does

- Add subjects with exam dates and difficulty level
- Generate a personalized day-by-day study schedule using AI
- Daily check-in: mark whether today's target was completed
- If a target is missed, the schedule automatically re-plans the remaining days
- Simple dashboard to track progress and upcoming deadlines

## Why this project

As a BSCS student juggling multiple subjects and exam dates, I wanted a planner that doesn't just create a static schedule but actually adapts when real life gets in the way — which is where AI collaboration comes in.

## Tech stack

- React (Vite)
- Tailwind CSS
- JavaScript
- Claude API for schedule generation and re-planning logic
